PennOS: UNIX-Like Operating System

OS written in C that runs over Linux. Includes a shell which has piping, redicrection, and process creation, a kernel (POSIX API) that uses the ucontext library to provide multithreading to users, and a FAT filesystem with a user-level API to create files and manage permissions.
